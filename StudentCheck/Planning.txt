Requirements:

Plan and build a Console application that asks a user for their name 
and their age. If their name is Bob or Sue, address them as Professor.
If the person is under 21, recommend they wait X years to start the class.


Psuedo Code:

ask for name: {firstName}
ask for age: {age}

if {firstName} = Bob or Sue
	{formalName} = "Professor {firstName}"
else 
	{formalName} = {firstName}

if age < 21
	Please wait {21-age} years to begin this class, {formalName}.
else
	Welcome to the class, {formalName}.
